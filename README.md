# Corre-Vejigante
Infinite runner for mobile with elements of puertorrican culture.
UNDER DEVELOPMENT

PRE-ALPHA VERSION
Unreal Engine 4.19 or higher required.
Updates for music and bugs coming later.

-The track spawns in the middle of the screen.
